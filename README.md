# TransitSync Seoul - GitHub Pages PWA

이 폴더의 파일들을 GitHub Repository 루트에 그대로 업로드하면 됩니다.

## 포함 파일
- index.html
- manifest.webmanifest
- service-worker.js
- icons/
- .nojekyll

## GitHub Pages 설정
1. GitHub에서 새 Repository 생성
2. 이 폴더 안의 파일 전체 업로드
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / root 선택
6. Save
7. 생성된 `https://아이디.github.io/저장소명/` 주소로 접속

## iPhone 설치
1. iPhone Safari에서 GitHub Pages 주소 접속
2. 공유 버튼
3. 홈 화면에 추가
4. 앱처럼 실행
